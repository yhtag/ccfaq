Card+ Registration User Guide
            
     Card + Application အားအသုံးပြုမည်ဆိုပါက ရှေးဦးစွာ Register Now မှ တစ်ဆင့် Register ပြုလုပ်ရမည်ဖြစ်ပါသည်။

      Card + Application တွင်စစ်ဆေးနိုင်သော ကတ်အမျိုးအစားများ - Prepaid Card and Credit Card

      Card Deactivate ပြုလုပ်ထားခြင်း/ Card Activate မပြုလုပ်ရခြင်းတို့ဖြစ်ပါက Card+ Application တွင် Card အားမမြင်တွေ့ရပါ။

      Card+ Application အား ပထမဦးဆုံး Register ပြုလုပ်ထားသည့် Email ID ဖြင့်အသုံးပြုနိုင်မည်ဖြစ်ပြီး Email Address ပြောင်းလဲအသုံးပြု၍မရပါ။

      Card+ မှ တစ်ဆင့် ကိုယ်တိုင် Deactivate/Hide Card ပြုလုပ်ထားပါက ကိုယ်တိုင် Activate/ Unhide ပြန်လည်ဆောင်ရွက်နိုင်ပါသည်။

      Deactivate / Hidden Card /Expired Card များအား More>> Organize Cards တွင်ဝင်ရောက်ကြည့်ရှုနိုင်ပါသည်။

      Digital Visa/Master Card အားလျှောက်ထားလိုပါက Card+ >>Home Page>> Apply Now မှလည်း လျှောက်ထားနိုင်သလို

     More>> Apply မှ တစ်ဆင့်လျှောက်ထားနိုင်ပါသည်။

      CB Pay ရှိ Card+ တွင် Card Detailed/ Statement /History များစစ်ဆေးနိုင်ပါသည်။

      မတူညီသည့်အတွက်မပေါ်နေခြင်းဖြစ်ပြီး Card Number ရိုက်ထည့်၍ report ပြုလုပ်နိုင်ပါသည်။Card မြင်တွေ့ရရန်

      Registration Process

                                                

      - Click Register Now (Registration with two ways)

      - 1.(i) Type Card Number (or) tab Camera icon to scan Card Number

      -1.(ii) Type Father Name and "Confirm" Process

      - Other Option >2.Registration with ID

      - 2.(i) Type ID Type

      -2.(ii)Type ID Number

      -2.(iii)Choose Nationality

      -2.(iv)Father Name and tab "Next"

      - After That "Mobile Verification Code " Phone Number ထဲသို့ OTP code ရောက်ရှိလာမည်ဖြစ်ပြီး OTP code အား
ထည့်သွင်းပေးရမည်ဖြစ်ပါသည်။

      - "Email Verification " အဆင့်သို့ရောက်ရှိပြီး Card လျှောက်ထားစဉ်ကဖြည့်သွင်းထားသောEmail ထဲသို့ Verification Link
ပေးပို့ပေးမည်ဖြစ်ပါသည်။

      - Email ထဲသို့ ရောက်ရှိလာသော မေးလ်အားဝင်ရောက်၍ Verify ပြုလုပ်ပြီးနောက် "Containue Registration" အားနှိပ်ပါ။

      - ထို့နောက် Password Create ပြုလုပ်ပေးရမည်ဖြစ်ပါသည်။

      - Password Create ပြုလုပ်ပါက Password အား (i) at least 8 characters

              (ii) Include a combination of upper case alphabets ,[e.g. A, B, C] and lower case alphabets [e.g. a, b, c]

              (iii)numbers [e.g. 1, 2, 3].

              (iv)Include at least one special character [@, #, $, %, *]

              Enter the Confirm Password and confirm

      - Password Create ပြုလုပ်ပြီးပါက MPin Create ပြုလုပ်ပေးရမည်ဖြစ်ပါသည်။

        MPin Create ပြုလုပ်ပါက -(i)Set the 6-digit mPIN

        Enters the same mPIN to confirm

     - MPin Create ပြုလုပ်ပြီးပါက Touch ID Create ပြုလုပ်ပေးပါ။

     - Mobile Phone ၏ Fingerprint sensor တွင် Fingerprint ယူရန်အတွက် ဆောင်ရွက်ပြီး Set Touch ID အားနှိပ်ပေးပါ။

     - Registration Process ပြီးဆုံးပါက Log in နေရာတွင် Email Address နှင့် Register ပြုလုပ်စဉ်က ဖြည့်သွင်းထားသော
Password ရိုက်၍သော်လည်းကောင်း၊ Fingerprint ဖြင့်သော်လည်းကောင်း Login ဝင်ရောက်နိုင်ပြီဖြစ်ပါသည်။
