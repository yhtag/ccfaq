 ဝန်ဆောင်ခနှုန်းထားများ

     - ယခုအချိန်အထိ အခမဲ့ဆောင်ရွက်ပေးပါသည်။
 လုပ်ဆောင်ချက်များ

  1.Own Account Transfer

    - မိမိအကောင့်အချင်းချင်းထဲသို့ ငွေလွှဲခြင်းဖြစ်သည်။(Only CB to CB)

 2. Transfer to Other Account/Card

    - အခြားအကောင့်/ကဒ်များသို့ ငွေလွှဲခြင်းဖြစ်သည်။(Only CB to CB)

 3. Benificiary Payment

   - မိမိ၏စာရင်းရှင် စာရင်း (သို့မဟုတ်) ငွေစုဘဏ် စာရင်း ၊နိုင်ငံခြားငွေ စာရင်းများမှ တဆင့် မိမိပေးချေ

    လို သောသူ၏ စာရင်းရှင်စာရင်း ၊ ငွေစုဘဏ်စာရင်း၊နိုင်ငံခြားငွေ စာရင်း များသို့ ပေးချေ နိုင်ခြင်း။

 3. Account Statement

   - မိမိတစ်နေ့တာ အသုံးပြုထားသည့် ငွေအဝင်အထွက်စာရင်းများကိုကြည့်ရှုခြင်း။

 4. P2P Money Transfer Service

   - Mobile Banking ဖွင့်ထားသူမှ အကောင့်မရှိသူများသို့ ငွေလွှဲခြင်းဖြစ်သည်။

    - ငွေလွဲှမည်ဆိုပါက လက်ခံမည့်သူ၏

      (၁) အမည်

      (၂) မှတ်ပုံတင်နံပါတ်

      (၃) ဖုန်းနံပါတ်

    - ငွေလက်ခံမည့်သူ၏ဘက်မှ

      (၁) မှတ်ပုံတင်

      (၂) Security Code No.

      (၃) Reference No.

      (၄) ငွေပမာဏ

      (၅) ငွေလွဲှပေးသည့်နေ့တို့ကို သိရမည်။

    - CB Bank နှင့် နီးစပ်ရာ Mobile Agent များတွင် ထုတ်ယူနိုင်သည်။

    - ဝန်ဆောင်ခ ၁၀၀ကျပ်လျှင် ဝ.၁၅%

 5. Beneficiary Registration

   - မိမိငွေပေးချေလိုသော သူ၏အမည်နှင့်စာရင်းအမှတ်အား မှတ်ပုံတင်ပေးခြင်း။မိမိဖုန်းတွင် ကိုယ်တိုင် ပြုလုပ်နိုင်သည့်

     အတွက် တခြားသူများ မသိရှိအောင် ငွေပေးချေနိုင်ခြင်း၊

 6.  Beneficiary Deletion

   - မိမိငွေပေးချေလိုသော သူ၏အမည်နှင့်စာရင်းအမှတ်အား မှတ်ပုံတင်ခြင်းမှပယ်ဖျက် နိုင်ခြင်း။

 7. Standing Order Registration

   - မိမိ၏စာရင်းရှင် စာရင်း (သို့မဟုတ်) မိမိငွေစုဘဏ်စာရင်းမှ မိမိပေးချေလိုက်သော တခြားသူ များအား ငွေပေးချေ

    လိုက်သောကာလအပိုင်းအခြား၊ ငွေပေးချေလိုက်သော ပမာဏ ၊ ငွေပေးချေ ခြင်းအား နောက်ဆုံးရပ်ဆိုင်းမည့်နေ့ကို

     အသေးစိတ် သိရှိနိုင်ပါသည်။
