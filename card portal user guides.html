    Card Portal မှ တစ်ဆင့် ကိုယ်တိုင်ဆောင်ရွက်နိုင်သော ဝန်ဆောင်မှုများ

   1.   Wallet Transfer

    - မိမိ၏ ကတ်တွင် Currency နှစ်မျိုး (သို့) ၃မျိုးပါရှိပါက Currency အချင်းချင်း Transfer ပြုလုပ်ခြင်း

     - Cardportal Login >>Home >> Fund Transfer >> Choose Wallet Transfer >> Choose Card Number >>

      Choose Currency and done process.

                                                                

   2.   Card to Card Transfer

    - Visa Card မှ Master Card သို့သော်လည်းကောင်း၊ Master Card မှ Visa Card သို့သော်လည်းကောင်း၊

    Visa Card အချင်းချင်း(သို့) Master Card အချင်းချင်းငွေလွှဲဆောင်ရွက်နိုင်ခြင်း။

     - Cardportal Login >>Home >> Fund Transfer >> Choose Card to Card Transfer >>

     Choose Card Number and Writes Transfer Card Number >> write amount and done process.

     - ဝန်ဆောင်ခများ

     -10% per transactions

     - min - 10 USD to 200 USD / 300 USD per day

                                                                              

   3. - Change Pin/Request Pin

     Card Portal မှ တစ်ဆင့် ဝန်ဆောင်ခ ပေးရန်မလိုအပ်ပဲ ကိုယ်တိုင် Pin Number အသစ်ထုတ်ယူခြင်း။

     Card Portal Login >> Home >> Request Pin/Change Pin>> then process.

                                                                

   4. - Card ပိုင်းဆိုင်ရာလုံခြုံမှုရှိစေရန်အတွက် ကိုယ်တိုင်ကတ်အား ယာယီပိတ်သိမ်း/ ပြန်လည်ဖွင့်နိုင်ခြင်း။

     Card Portal Login >>Home>> Card Block and UnLock

     Card Block ပြုလုပ်နိုင်သော အကြောင်းအရာများ - Lost Card (LC), Stolen Card (SC), Temporary Block (TB),

     Travelling (TR)

    အထက်ပါ Block ပြုလုပ်နိုင်သည့် အကြောင်းအရာ(၄)မျိုး အနက်မှ ကတ်ပျောက်ခြင်း ၊ ကတ်အခိုးခံခြင်း အကြောင်းအရာဖြင့်

    Block လုပ်ထားပြီး ၊ Unblock လုပ်ပေးရန် ကတ်ပိုင်ရှင်ကိုယ်တိုင်သော်၎င်း ၊ အခြားတစ်စုံတစ်ဦးမှ သော်၎င်း

    Call Center သို့ ဆက်သွယ်လာပါက ကတ်ပိုင်ရှင် ဟုတ် ၊ မဟုတ် Card Information အချက်အလက်များ

    သေချာတိုက်စစ် စစ်ဆေးပြီး Unblock လုံးဝမပြုလုပ်ပေးဘဲ ကတ်ပိုင်ရှင် အစစ်အမှန်အားဘဏ်သို့ကတ်ကိုင်ဆောင်သူ

    ကိုယ်တိုင်ကတ်အသစ် ပြန်လည်လျှောက်ထားနိုင်ပါကြောင်း အသိပေးလမ်းညွှန်ပေးပါရန်။

    ယာယီပိတ်ထားလိုခြင်း ( Temporary Block ) ၊ ခရီးသွားနေစဉ်အတွင်း ခတ္တယာယီပိတ်ထားလိုခြင်း( Traveling )

    အကြောင်းအရာဖြင့် Block ပြုလုပ်ထားပါကcard portal မှတစ်ဆင့်ကတ်ပိုင်ရှင်ကိုယ်တိုင်Unblock ပြန်လည်
